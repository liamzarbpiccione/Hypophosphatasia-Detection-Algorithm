# Hypophosphatasia-Detection-Algorithm
A customisable algorithm capable of searching multiple excel files based on adjustable bone profile analytes.

Should you utilise this algorithm in your use kindly reference the original publication/thesis.
Please feel free to make use/edit the code to best suit your needs.

# Usage

the main .py file is the code used for the algorithm while the .exe is ready to use. This brings up a menu to input one or more excel files.

following this a second menu prompts the user to change desired analytes based on preference. 
This is ideal considering the many different reference ranges used in different hospitals/assay platforms.

following this a summary statistics page will be generated and the final excel file will be outputted to the desktop
results are timestamped to help keep track of individual results.
